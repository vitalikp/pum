Pum
======
packages update manager

# Install
 - download [pum](https://github.com/vitalikp/pum/archive/master.tar.gz);
 - unpack archive and go into directory;
 - create **build** directory;
 - run commands:
```
	# cd build
	# cmake ..
	# make
	# sudo make install
```
